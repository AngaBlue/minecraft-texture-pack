## AngaBlue's Edit
A 32x resource pack designed to be used with the [Faithful 32x resource pack](https://faithful.team/tag/faithful/).
## Features

 - Clear GUI w/ Blue Theme
 - Custom Font
 - Low Fire
 - Low Shield
 - Bow Charge Indicator
 - Cloudy Skybox
 - Better Sun + Moon
 - Greener Foliage
 - Coloured Item Names
 - Clearer Water
 - Removed Pumpkin Overlay
 - Custom Splashes
